infomark work

start a worker

Synopsis

Starts a background worker which will use docker to test submissions. Can be used with the flag “-n” to start multiple workers within one process.

infomark work [flags]

Options

  -h, --help         help for work
  -n, --number int   number of workers within one routine (default 1)
